如何使用Hexo+GiuHubPage自定义自己的博客

August 25, 2019 - views

经过这些天的再次摸索，终于有了自己的博客，原本是想直接采用别人的主题，但是还是想自己创建一个。然后我就去寻找关于“如何自定义创建主题”的文章，索性寻找到了一些线索，最后我决定还是仿照自己喜欢的页面。

让我们开始吧

首先是基本的几连，相信你看官方文档就能明白的。没错就像是这样，前提是在命令行中输入哦：

正如官方文档所言，这只能在本地查看，并不能放到 github page 上，通过网址浏览。此时需要把写的文章，转换成静态的网页，并转发到相应的仓库中。大致三步，修改配置文件，生成静态文件，上传。

此时就能在 git 上查看到上传的文件了，当然此时的页面主题是官方的主题，接下来我们就来创建属于自己的主题。

获取自己的主题

正如官方文档所说，你可以采用他人的主题，一般别人的主题都会有 readme 介绍的。步骤大致两步，首先下载主题代码，然后配置相关的项，有的主题配置的时候需要安装额外插件，总之有 readme 介绍的就放心使用。

创建自己的主题的话就需要有 html + css + js + ejs 的知识，相信对你来说 html + css + js 是手到擒来的，ejs 的话是一个模板语言，大概就是可以在 html 中插入 js，说起来好像和 js 一样。没错，其实差不多，但是还是需要先知道它的语法糖，搜官方文档就完事了。

相信你看过官方文档之后，心里就有眉目了，想跃跃欲试了，在这之前，不如先看看几篇文章。

  • ejs模板引擎
  • Create an Hexo Theme - Part 1: Index
  • Create an Hexo Theme - Part 2: Other Pages 09 March 2016
  • Create an Hexo Theme - Part 3: Comments, Analytics and Widgets
  • Hexo主题开发

看过这级片文章，聪明的你应该已经会动收制作了。此时你可以想自己想要一个什么样的主题，那我来说，我的话就想存放一个自己笔记文字的地方，所以我选择了一个比较喜欢的博客，rauchg.com。如果你还没拿定主意，那么可以去搜寻属于你的主题，这里推荐两个地方，一个是 Hexo Top 10 Popular Themes，另一个是 有哪些好看的 Hexo 主题？。

选择好了主题那就开始制作吧。
